# ocf-optical-flow-tvl1

This is an implementation of the OpticalFlowTVL1 model for the work on cloud forecasting with [Open Climate Fix](https://openclimatefix.org/).
It uses [`cloudcasting`](https://github.com/alan-turing-institute/cloudcasting) to validate models against EUMETSAT data.

## Usage

```bash
pip install -e .
```

```bash
cloudcasting validate
```
